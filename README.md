# World Chat

Глобальный чат в реальном времени. Все сообщения видны всем посетителям.

## Возможности

- Мгновенные сообщения через WebSocket
- Темная и светлая тема
- Отправка фото, видео и файлов
- Голосовые сообщения
- Опросы и викторины
- Ответы на сообщения, реакции, редактирование
- Markdown в тексте (жирный, курсив, код, спойлеры)
- Поиск по сообщениям
- Закрепленные сообщения
- Админ-панель с модерацией
- Быстрые сообщения для админа
- Экспорт истории чата
- Поддержка русского и английского языков

## Технологии

- **Frontend:** HTML, CSS, JavaScript (Vercel)
- **Backend:** Node.js, WebSocket, Express (Render)
- **Хранение:** JSON (файл на сервере)
- **Верификация:** EmailJS + Mail.ru SMTP

## Установка

1. Клонировать репозиторий
2. Задеплоить `backend/` на Render
3. Задеплоить `frontend/` на Vercel
4. Настроить переменные окружения на Render
5. Настроить EmailJS для верификации
